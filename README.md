# japerribay.github.io

Repositorio preparado para servir una página estática con GitHub Pages desde la raíz del proyecto.

## Estructura

```text
.
|-- .nojekyll
|-- 404.html
|-- README.md
|-- assets
|   `-- css
|       `-- styles.css
`-- index.html
```

## Uso con GitHub Pages

1. Sube el contenido al repositorio `japerribay.github.io`.
2. En GitHub, activa GitHub Pages usando la rama principal y la carpeta raíz (`/`).
3. GitHub publicará `index.html` como página de inicio del sitio.
